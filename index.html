Teoria de Áutomatas

Gramática

{ anbn+m | n y m >= 0 }

Aceptada:

  • bbbb
  • aabb
  • aaabbbbb

    • No aceptada:

  • aab
  • abba
  • bbba

    • Instrucciones

    • Primeramente, ingresa una cadena de caracteres en el campo de texto.
    • Luego, presiona el botón "Guardar" para guardar la cadena en el campo de texto.
    • Finalmente, presiona el botón "Validar" para validar la cadena.
    • En caso de desear borrar la cadena, presiona el botón "Borrar".

    Mensajes de alerta

    • Si la cadena es válida, se mostrará un mensaje de éxito.
    • Si la cadena no es válida, se mostrará un mensaje de error.
